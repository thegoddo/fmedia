# fmedia

A Torrent scraper to scrape content from your terminal.

## Description

A python scraper to scrape rarbg website and stream your torrent
with vlc and peerflix.

## how to use

- Install Peerflix

```bash
npm install -g peerflix
```

- Install VLC

```bash
# Fedora/RedHat
dnf install vlc

# Ubuntu
apt install vlc

# Windows
winget install vlc
```

- Run this script

```bash
python main.py
```
